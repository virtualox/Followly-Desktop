**Analyze your Instagram followers and following with ease.**

Track who unfollowed you, see new followers, and export your data in multiple formats. Available for Windows, macOS, and Linux.

## 🚀 How to Use

1. **Install Browser Extension:**
   - Download from Chrome Web Store, Safari Extensions, or Firefox Add-ons
   - One-click export of Instagram followers and following data

2. **Import in Desktop App:**
   - Launch Followly Desktop
   - Click "Import CSV"
   - Select exported CSV files from the browser extension

3. **Analyze:**
   - View who unfollowed you
   - See new followers
   - Compare changes over time

4. **Export:**
   - Choose format (CSV, HTML, Excel, PDF)
   - Select destination folder
   - Open report

## 🔒 Privacy & Security

- ✅ **No data collection** - Your data stays on your device
- ✅ **Secure storage** - Credentials stored in OS keychain
- ✅ **OAuth only** - No password storage

**License storage:**
- Windows: Credential Manager
- macOS: Keychain
- Linux: Secret Service API
