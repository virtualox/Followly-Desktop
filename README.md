# Followly Desktop

**Official releases for Followly Desktop**

Native desktop application for analyzing Instagram followers and following data. Import CSV files, compare snapshots, and export detailed reports.

---

## 📦 Platforms

Download the latest version from the [Releases](https://github.com/virtualox/Followly-Desktop/releases) page.

### Windows (Available Now)
- **MSI Installer** (Recommended) - Full installer with graphical wizard
- **Portable ZIP** - Single-file executable, no installation required
- Both formats are **self-contained** - no .NET runtime installation needed!

### macOS (Coming Soon)
- Native macOS application for macOS 12 Monterey and later
- Universal binary (Apple Silicon + Intel)

### Linux (Coming Soon)
- Support for major distributions (Ubuntu, Fedora, Arch)
- AppImage and .deb packages

## 🚀 Installation

### Windows

**MSI Installer (Recommended):**
1. Download `Followly-X.X.X-Setup.msi` from [Releases](https://github.com/virtualox/Followly-Desktop/releases/latest)
2. Run the installer
3. Follow the installation wizard
4. Launch from Start Menu or Desktop shortcut

**Portable Version:**
1. Download `Followly-X.X.X-Portable-x64.zip` from [Releases](https://github.com/virtualox/Followly-Desktop/releases/latest)
2. Extract to any folder
3. Run `Followly.exe`

No installation or admin rights required!

### macOS (Coming Soon)

**DMG Installer:**
1. Download `Followly-X.X.X.dmg` from Releases
2. Open the DMG file
3. Drag Followly to Applications folder
4. Launch from Applications or Spotlight

### Linux (Coming Soon)

**AppImage:**
1. Download `Followly-X.X.X.AppImage` from Releases
2. Make executable: `chmod +x Followly-X.X.X.AppImage`
3. Run: `./Followly-X.X.X.AppImage`

**Debian/Ubuntu (.deb):**
```bash
sudo dpkg -i Followly-X.X.X.deb
sudo apt-get install -f
```
